proc makeDistDom(size: int)¶
Uses the MyDmap config param in ServerConfig.chpl::

if MyDmap == 0 {return (type CyclicDom(1,int(64),false));}

if MyDmap == 1 {return (type BlockDom(1,int(64),false,unmanaged DefaultDist));}

Arguments:size : int – size of domain

Note: if MyDmap does not evaluate to 0 or 1, Cyclic Distribution will be selected. Cyclic Distribution is currently not fully supported. Note 2: MyDmap is by default set to 1 in ServerConfig.chpl
