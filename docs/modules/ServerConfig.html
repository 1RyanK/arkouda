ServerConfig¶

Usage

use ServerConfig;

arkouda server config param and config const

config const v = true¶

verbose flag

config const ServerPort = 5555¶

port for zeromq

config param arkoudaVersion = "0.0.9pre"¶

arkouda version

config param MyDmap = 1¶

configure MyDmap on compile line by “-s MyDmap=0” or “-s MyDmap=1” 0 = Cyclic, 1 = Block, Cyclic may not work now haven’t tested it in a while BlockDist is the default now
